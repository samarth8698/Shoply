# 🛒 Shoply – Full Stack E-Commerce Application

Shoply is a **Full Stack E-Commerce Application** developed using **React.js**, **Spring Boot**, **Spring Data JPA**, and **MySQL**. The application provides a complete online shopping experience with product browsing, wishlist management, order processing, and REST API integration.


# 🚀 Project Overview

The project follows a **three-tier architecture**:

* **Frontend:** React.js + Vite
* **Backend:** Spring Boot + Spring Data JPA
* **Database:** MySQL

The frontend communicates with the backend through REST APIs, while the backend manages business logic and database operations.

# ⚙️ Installation

## 1️⃣ Clone Repository

bash
git clone https://github.com/samarth8698/Shoply.git


## 2️⃣ Backend Setup

bash
cd Backend/Shoply-Backend


Run the Spring Boot application.

Backend URL

text
http://localhost:8080


## 3️⃣ Frontend Setup

bash
cd Frontend/shopping
npm install
npm run dev


Frontend URL:-
http://localhost:5173


## 4️⃣ Database:-

* Create a MySQL database.
* Import the SQL file from the "Database" folder.
* Update database credentials in "application.properties".
