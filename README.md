# Shoply - Full Stack E-Commerce Application

## 📌 Overview
Shoply is a Full Stack E-Commerce web application built using React.js, Spring Boot, and MySQL. Users can browse products, manage their wishlist, place orders, and track orders through a simple interface.

## 🛠 Tech Stack
### Frontend
- React.js
- Vite
- React Router
- Axios

### Backend
- Spring Boot
- Spring Data JPA
- REST API

### Database
- MySQL

## ✨ Features
- User Authentication
- Product Listing
- Product Details
- Wishlist
- Shopping Cart
- Order Placement
- Order Tracking
- Admin Dashboard

## 📂 Project Structure
Shoply/
├── Frontend/
├── Backend/
└── Database/

## 🚀 Installation

### Frontend
cd Frontend/shopping
npm install
npm run dev

### Backend
cd Backend/Shoply-Backend
mvn spring-boot:run

### Database
Import Database/shoply.sql into MySQL.
